# NKnife

## 序言
这是一个基于.Net的综合开发库，集成了作者在研发工作过程中的大量开发经验。虽然开源，但是作者还是比较偷懒，技术上也没有达到统一的能力，所以本开发库仅适用于同行参考，不建议生产使用。
